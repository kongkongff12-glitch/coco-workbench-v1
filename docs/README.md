# COCO 客户工作台 重构 v1.1

打开 `app.html` 即可使用；`index.html` 是默认入口，`workbench.html` 同步为同一版。

- 定位：客户工作 OS + 自动白板生成器，优先替代每周人工白板。
- 核心逻辑：一次录入，多处呈现，重点先行，原文和源文件只作为可展开依据。
- 白板结构：日期、每周一句话判断、最近讨论重点、按重要度排序事项、双方待办、客户心理变化、下次会议讲法。
- 主导航：全客户总览、客户白板、工作动态、客户事项、交付与内容库、数据中心、录入。
- COCO 当前重点：SuperAI 活动资产沉淀、GA / Similarweb / PostHog 数据闭环、客户案例包装、视频脚本复用。
- 会议档案：展示完整会议分析与 Action；转录原文只显示内部语料状态，不直接渲染。
- 社媒成果：已发布且表内有 URL 的内容显示可点击链接。
- 飞书社媒看板：https://my.feishu.cn/wiki/KGJJwc6iOiFMeokvIJrcQM81n1d?sheet=1Jbfs5
- 录入同步：页面会优先提交到 `/api/entries`；GitHub Pages 下无后端，会本地保存；迁到 Vercel 并配置环境变量后即可自动写回仓库并全员同步。
