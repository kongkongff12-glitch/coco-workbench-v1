# COCO 周五白板生成逻辑

## 正确时间顺序
1. 每周五开会前，先持续更新白板。
2. 周五会议直接使用白板过进展、问题、数据、材料和下一步。
3. 会议文本是会后的记录，不能反向混进当周会前白板。
4. 会后根据会议文本、群聊、每日产出与数据，生成下一周周五白板。

## 白板应有结构
1. 会议日程总览：单列呈现，本周要过什么。
2. 上周重点回顾：只保留最重要的结果和变化。
3. 客户最近新提出或集中讨论的问题：按问题列出。
4. 我方处理方式/解决方案：对应到负责人、材料、数据或排期。
5. 明细区：视频、SEO、数据、社媒、素材库、交付节点。
6. 证据区：每个判断关联群聊日期、会议文本或素材文件。

## 设计原则
- 前面看重点，后面看明细。
- 客户展示页少废话，避免心路历程。
- 所有数据、判断、客户心理变化都要可追溯。
